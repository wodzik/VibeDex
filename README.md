## VibeDex
[VibeDex](https://github.com/wodzik/VibeDex) is an alg trainer based on the original [Cubedex](https://cubedex.app).  
It helps you drill, time, and master Rubik's Cube algorithm sets like OLL and PLL, building them into your muscle memory more quickly and effectively.
